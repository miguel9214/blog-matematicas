# Instrucciones para las fotos del equipo

Coloca las fotos de los integrantes del equipo en esta carpeta con los siguientes nombres:

1. `luis-alfonso.jpg` - Luis Alfonso Núñez Carpio
2. `neila.jpg` - Neila Pineda Cañizares
3. `yuleinis.jpg` - Yuleinis Urquiza Pimienta
4. `yonathan.jpg` - Yonathan Peinado
5. `skarlyn.jpg` - Skarlyn Gabriela Zambrano Tabares

## Recomendaciones:

- Usa fotos cuadradas (o se recortarán como círculos)
- Tamaño recomendado: 300x300px o superior
- Formato: JPG o PNG
- Peso: Preferiblemente menos de 200KB cada una

## Foto por defecto:

Si no tienes una foto disponible para algún integrante, se mostrará una imagen por defecto.
